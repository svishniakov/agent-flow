Routing

Enough process. No more.

Agent Flow reads scope, risk, project memory, and required evidence. Then it chooses a compact route instead of treating every task like a full orchestration run.

1

Quick check

One command, short answer, small diagnostic, no trace directory.

2

Bugfix or feature

Reproduce, patch, verify. Keep edits narrow and evidence fresh.

3

Docs and design

Shape product docs, README, PRD, UI route, or approved design source.

4

Release risk

Traceable run for CI, deploy, auth, secrets, external systems, or high-risk checks.
